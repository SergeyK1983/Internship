Содержание

  • /api/v1/create/ - POST, добавление записи о перевале в БД cо статусом "Новый"
  • /api/v1/submitData/{id} - GET, просмотр информации о перевале
  • /api/v1/submitData/{email}/ - GET, просмотр всех записей пользователя
  • /api/v1/update/{id} - PUT, изменение статуса записи модератором
  • /api/v1/submitData/update/{id} - GET, PUT, изменение (просмотр) записи пользователем
  • /api/v1/perevals/ - GET, просмотр всех id записей
